# TaxIP 2.0 - Dashboard Administrativo

## Descripción

Dashboard moderno para la gestión de flota de taxis, desarrollado con Next.js 14, TypeScript, Tailwind CSS y shadcn/ui.

## Requisitos previos

- Node.js 18+
- Backend TaxIP 2.0 ejecutándose en `http://localhost:8000`

## Instalación

```bash
# Instalar dependencias
npm install

# Configurar variables de entorno
cp .env.local.example .env.local
# Editar .env.local con tus credenciales

# Iniciar servidor de desarrollo
npm run dev
